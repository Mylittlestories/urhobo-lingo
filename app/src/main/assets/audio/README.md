# Audio Assets

Place real verified audio files here.

## Structure
- `guidance/` for onboarding and guide narration
- `pronunciation/` for word/phrase clips
- `manifest.json` maps IDs to audio files

## Recommended recording spec
- speaker: real Urhobo-speaking woman
- format: mp3 or wav
- sample rate: 44.1kHz preferred
- clean quiet recording
- one clip per phrase/word

## Example filenames
- `guidance/welcome_intro.mp3`
- `pronunciation/hello_mighwo.mp3`
- `pronunciation/good_morning_mighwo_royibo.mp3`

## Important
Current files referenced in the manifest are placeholders. Replace them with real recordings.
