### Notes

- It links related Urhobo resources and external audio sources.
- I did not verify a female speaker from these sources.

## Recommended acquisition strategy
1. Source a real Urhobo-speaking woman locally
2. Record a starter set of 100-300 phrases/words
3. Save as bundled assets in `app/src/main/assets/audio/`
4. Update `manifest.json`
5. Use app playback for pronunciation instead of generic TTS
