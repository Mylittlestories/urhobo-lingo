# Voice Strategy for UrhoboLingo

## Current reality
There is no confirmed mainstream, production-ready female Urhobo TTS voice integrated in this project today.

## Recommended professional approach
### Best option
Record a real Urhobo-speaking woman for:
- pronunciation clips
- onboarding guide voice
- phrase playback
- lesson examples

### Why
A real native or fluent speaker gives better authenticity than forcing another language TTS engine to approximate Urhobo.

## Good implementation options
1. **Pre-recorded audio assets**
   - Save `.mp3` clips by word/phrase
   - Best for quality and authenticity
2. **Custom TTS later**
   - Build or train a voice if you gather enough speech data
3. **Temporary fallback**
   - Use English female TTS for guide narration only
   - Do not pretend it is an authentic Urhobo voice

## App recommendation
- Use native recorded female Urhobo audio for pronunciation
- Use optional English female narrator for app guidance until a true Urhobo synthetic voice exists
