# Store Listing Draft

## App Name
UrhoboLingo

## Short Description
Learn Urhobo in a fun, playful way with lessons, quizzes, vocabulary, and translation.

## Full Description
UrhoboLingo helps beginners and families learn Urhobo through short interactive lessons, a gamified learning path, vocabulary review, progress tracking, and an English ↔ Urhobo translator.

Features:
- playful lesson flow
- quick quizzes
- vocabulary practice
- saved words
- progress, streaks, hearts, and XP
- translation support

## Keywords
Urhobo, learn Urhobo, language learning, translator, African languages, vocabulary, quiz
